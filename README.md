# vue-searchInput-demo

技术栈：webpack + vue-cli + vue-resource

一个简单的搜索栏demo，调用的是百度的接口，使用vue-cli脚手架自动生成vue项目，使用Webpack对项目进行打包处理，使用vue-resource发送请求，并拿到返回数据进行显示。按回车键或点击搜索按钮都可以进行搜索。demo中对搜索栏进行了组件封装，可以进行复用。
